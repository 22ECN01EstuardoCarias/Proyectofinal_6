" Etiquetas y Atributos en HTML "

' Principales/Básicas '

html: Define el documento HTML.
<html>..Inicio y Fin..</html>

head: Contiene metadatos/información del documento.
<head>..Encabezados..</head>

title: Titulo de la página (aparece en la pestaña del navegador).
<title>..Pestaña del navegador..</title>

body: Contiene la información de la página.
<body>..Cuerpo de la página web..</body>

h1 a h6: Tamaños de Títulos y Subtítulos de mayor a menor nivel.
<h1 a h6>..Títulos/Subtítulos..</h1 a h6>

p: Párrafos o texto.
<p>..Textos..</p>

br: Saltos de Línea.
<br>..Siguiente línea..</br>

hr: Línea horizontal de separación.
<hr>..Una Separación/División..</hr>

a: Hipervínculo.
<a>..Hipervínculo..</a>

img: Vínculo pra colocar una Imagen.
<img>..Imágenes..</img>

b: Colocar textos en negrita.
<b>..Negrita..</b>

i: Colocar textos en cursiva.
<i>..Cursiva..</i>

div: Contenedor en bloque sin estilo.
<div>..Bloque sin estilo..</div>

Section: Sirve para resumir los elementos en un grupo.
<section>..Resume elementos en grupo..</section>

Nav: La sección de navegación de una web.
<nav>..Sección de navegación web..</nav>

Span: Contenedor en línea, útil para aplicar estilos.
<span style="color:blue;">Texto</span>

Link: Enlace a recursos externos.
<link rel="stylesheet" href="estilos.css">

' Tablas '

Table: Creación de una tabla.
<table>..Tablas..</table>

Tr: Respectivamente a la fila de la tabla.
<tr>..Fila de tablas..</tr>

Td: Respectivamente a la celda de datos de la tabla.
<td>..Celda de Datos..</td>

Th o thead: Respectivamente correxponde a la celda del encabezado de la tabla.
<th o thead>..Celda de encabezado (negrita y centrada por defecto)..</th o thead>

Colspan: Atributo para la unión de columnas de la tabla.
colspan;..Unión de Columnas..

Rowspan: Atributo para la unión de filas de la tabla.
rowspan;..Unión de Filas..

tfoot: Agrupa el pie de la tabla.
<tfoot>..Totales u Observaciones..</tfoot>

' Listas '

ul: Lista NO ordenada.
<ul>..Con Viñetas..</ul>

ol: Lista ordenada.
<ol>..Numerado..</ol>

li: Elemento de lista
<li>..Dentro de ul u ol..</li>

Menu: Antiguamente para listas de menú.
<li>..En desuso pero válido..</li>

Article: Puede organizar listas en un arículo independiente.
<article>..Listas en artículo independiente..</article>

Type: Atributo que cambia el estilo de numeración o viñeta dentro de ul u ol.
Type;..Cambio de estilo de numeración o viñeta..

Reversed: Atributo que hace que la lista se muestre en orden descendente.
Reversed;..Orden Descendete..

' Formularios '

Form: Define unformulario para enviar datos.
<form action="procesar.php" method="post">...</form>

Input: Campo de entrada de datos ( texto, botón, chekbox, etc.).
<input type="text" name="nombre">

Label: Etiqueta descriptiva para un campo del formulario.
<label for="nombre">Nombre:</label>

Button: Botón para ejecutar acciones.
<button type="submit">Enviar</button>

Textarea: Área de texto de varias líneas.
<textarea>..Espacio para vairias líneas..</textarea>

Select: Lista e manera desplegable.
<select>..Opción despegable..</select>

Option: Para alguna opción dentro de la etiqueta 'select'.
<option>..Opción dentro de ´select..</option>

Meter: Representa un valor dentro de un rango.
<meter>..Como para una calificación..</meter>

Progress: Barra de progreso.
<progress>..Útil para formularios largos..</progress>
